
# L'objectif du projet:

Il s'agit d'un formulaire d'enregistrement pour les bénévoles d'une association. Un certain nombre d'informations sont demandées qui permettent de caractériser
les personnes. 

# Les parties:

1/ un formulaire d'inscription

Qui permet aux personnes d'entrer leurs données. 

2/ un fichier .csv de stockage des informations entrées dans le formulaire


3/ Une application de calculs et graphiques réalisés sur les données du csv.

Qui via la bibliothèque Streamlit va éditer des tableaux et graphiques sur les données.
